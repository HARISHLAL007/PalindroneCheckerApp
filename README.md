# PalindromeCheckerApp

## Objective

The objective of this project is to implement a Java program to check whether a number is a palindrome or not and to practice Git and GitHub workflow such as branching, committing, pushing, and merging using pull requests.

## Project Description

PalindromeCheckerApp is a simple Java application that checks whether a given number is a palindrome.
A palindrome number is a number that remains the same when its digits are reversed.

Example:
121 → Palindrome
123 → Not Palindrome
